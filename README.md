
<h1>Expense Tracker</h1>

<p>A simple Python OOP project to record and manage daily expenses.</p>

<h2>Features</h2>
<ul>
  <li>Add new expenses.</li>
  <li>Store title, amount, category, and date.</li>
  <li>View all expenses.</li>
  <li>Calculate total spending.</li>
  <li>Show category-wise summary.</li>
  <li>Handle invalid input safely.</li>
</ul>

<h2>OOP Concepts Used</h2>
<ul>
  <li>Classes and objects.</li>
  <li>Encapsulation.</li>
  <li>Methods.</li>
  <li>Input validation.</li>
</ul>

<h2>How It Works</h2>
<p>
  The program allows users to add expenses through a menu-driven interface.
  It validates the input, stores the details, and shows useful summaries like
  total spending and category-wise breakdown.
</p>

<h2>Tech Used</h2>
<ul>
  <li>Python</li>
  <li>Object-Oriented Programming (OOP)</li>
</ul>
